# sales-dashboard-powerbi
## Recommended Structure and Order
### 1.	Project Title / Headline

❄️ Business Analytics: Regional sales Insights Dashboard
A dynamic, interactive data visualization tool built to explore sales data focusing on regional comparisons, performance attributes, monthly track record, and payment mode.

### 2.	Short Description / Purpose
The Performance Analytics Dashboard is a visually engaging and analytical Power BI report designed to help users explore and compare over every key areas of the business, highlighting sum of amount by state, sum of quantity by category of products, profits earned in every month comparision, sum of amount by customer name so that gifts vouchers and other could be given to them for trust building, payments mode from widely usen to less usen, and at last sum of profit by sub category.
### 3.	Tech Stack
The dashboard was built using the following tools and technologies:<br>
•	📊 Power BI Desktop – Main data visualization platform used for report creation.<br>
•	📁 File Format – .pbix for development and .png for dashboard previews.

### 4.	Data Source
Source: Madha store economics data (excel sheet) during learning power bi.
Data on order id, quantity, amount, profit, category, sub-category, payment mode, ADV, order date, customer name and id, state and city.

-	States Filter Panel
An interactive slicer lets users filter all visuals by selected states, such as Maharashtra or Bihar.
-	Top Customers with Most Expenses (Bar Chart)
Bar chart ranks customers by their total expenses done during the period.
-	Area covered by product lines (Donut Chart)
Displays the distribution of product performance in a circumference like figure, easy to understand.
-	 Profit by Month (Dual Bar Graph)
Two bar visuals side by side: one for profit that cater to investment, another for loss for ROI comparision.
-	Elevation Stats (Bar Graph)
Compare the highest and lowest sub category of product performance.
-	Sub category of products (Bar Graph)
bar graph chart showing:

Tables
This helps users gauge product sales strength and capacity.
